¿Qué es el mouse?


El "mouse" (pronunciado "maus" en español) es un dispositivo de entrada que se utiliza comúnmente con las computadoras y otros dispositivos electrónicos para controlar el movimiento del puntero en la pantalla y realizar selecciones de elementos en la interfaz gráfica.



¿Qué compone a un mouse?



  • El mouse consta de dos botones principales

    El botón izquierdo se utiliza principalmente para hacer clic en elementos y realizar acciones como abrir programas, seleccionar texto y más.

    El botón derecho se utiliza generalmente para acceder a menús contextuales y realizar acciones específicas relacionadas con el elemento seleccionado.

  • La rueda de desplazamiento

    Se encuentra entre los botones, permite desplazarse hacia arriba o hacia abajo en documentos largos o páginas web sin tener que hacer clic repetidamente en las barras de desplazamiento.



El mouse se conecta a la computadora a través de un cable USB o de manera inalámbrica mediante tecnología Bluetooth o receptores USB inalámbricos.



¿Qué es el cursor?



El "cursor" es un símbolo visual que se utiliza en las interfaces gráficas de usuario de computadoras y dispositivos electrónicos para indicar la posición en la pantalla donde se realizará una acción cuando se presione un botón del mouse o del teclado.



  • Flecha: Utilizado para señalar y seleccionar elementos en la pantalla.

  • Mano: Se convierte en una mano cuando se coloca sobre un enlace o un elemento interactivo.

  • Indicador de Texto: Se utiliza en campos de texto para ingresar y editar texto.

  • Reloj de Arena o Círculo Giratorio: Indica que el sistema está procesando una tarea y se debe esperar.



El cursor es una parte esencial de la interacción usuario-computadora, ya que proporciona retroalimentación visual sobre las acciones que se pueden realizar y dónde se pueden realizar en la pantalla. Permite a los usuarios navegar, seleccionar, escribir y realizar una variedad de tareas de manera intuitiva en sus dispositivos electrónicos.
